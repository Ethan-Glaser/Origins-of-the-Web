Terminology

Internet
A set of protocols that when used together can connect systems of computers form anywhere in the world.
World Wide Web
Pages that are all linked together through a singular protocol on the Internet called HTTP. The Word Wide Web is not the Internet, but rather a subset of the Internet.
Hypertext
Text that is not linear, but rather when clicked in it brings you to more text or even pictures or videos.
Multimedia
Integration of different forms of media, like text, graphics, audio, or video, all in one place.
CLI
A text-based interface to enter in commands to a computer. CLI’s are typically used for fast pace performance.
GUI
A graphical interface to enter commands to a computer. GUI’s are typically used for ease of access to a user.
HCI
The study of how people interact with a computer. This science delves into the psychology of what we think when we are using the GUI and how we can make GUI easier and more beneficial for the user.
FTP
A protocol for the Internet that is used for transferring files over the Internet. Files are stored on an FTP server and can be accessed using an FTP client.
SSH
A protocol for the Internet that is used for secure communication. Data sent using SSH is encrypted so that even if your data intercepted it cannot be read.
HTTP
A protocol for the Internet that is used for transferring hypertext. HTTP is a critical protocol for the transmission of webpages and is what created the World Wide Web.
Telnet
A protocol for the Internet that is used for communication. Telnet is typically not used in the modern-day Internet due to its lack of security.
HTML
The programing language that is used for creating websites. Hypertext refers to the hyperlinks that the page contains, and markup language refers to the tags used while designing the site.
CSS
This is a web design language used for designing and formatting we pages. CSS helps web designers ensure that webpages look uniform and similar amongst the entire web site.
TCP
A protocol that has to do with the verifying delivery of the packets from one computer to another.
